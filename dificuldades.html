Difuculdades para criação do site

Falta de conhecimento de um ferramenta para compartilhar o projeto:

A nossa falta de conhecimento em ferramentas como o git e github, nos atrapalhou bastante no início do projeto. Uma forma que pensamos para solucionar esse problema era cada um fazer uma parte do site e depois juntar tudo... Bom, isso deu muito errado. Quando tentamos juntar o css de todo ficou uma bagunça. Para solucionar esse problema apredenmos a usar o básico de github, com isso, uma pessoa ficou encarregada de criar o repositório, e quando alguém fazia uma mudança mandava para essa pessoa e ela subia para o github. Assim quem fosse fazer outra parte já pegava a versão mais atual do projeto.

Problema no botão do menu

A ideia era criar um botão que com um clique abrisse um menu, essa parte foi muito difícil, primeiro tentamos usando lista não ordena, não deu certo porque não conseguimos deixar as li oculta e quando conseguimos quando passava o mouse em cima do botão abria, mas quando ia clicar em um li sumia, depois o Arthur usou javascript oque deu certo, mas apareceu muitos problemas como o posicionamento e em alguns momentos não funcionavam, com muito tempo de pesquisa na internet descobrimos uma forma de fazer usando div e no css usar a propriedade hover.

Problema com a orientação da logo

Enquanto estávamos com dificuldades no botão do menu, tivemos outro problema relacionado com a orientação da logo, depois de uns 3 dias tentando solucionar sem sucesso entramos em contato com o professor e ele nos ajudou a solucionar o problema.

layout

A princípio a ideia era fazer um layout como sendo vários blocos, no entanto, esse layout estava dando muitos problemas, então decidimos mudar para um layout com modelo de lista, pois, é mais fácil de fazer e muito mais simples para deixa-lo responsivo.
